# Proyecto Web de Sushi para Restaurantes Buffet

## Descripción
Este proyecto consiste en el desarrollo de una aplicación web para un restaurante de sushi, creada con el framework Angular.
La aplicación permite a los usuarios visualizar el menú, conocer información del restaurante y realizar interacciones básicas propias de una web moderna, ofreciendo una experiencia de usuario fluida, rápida y responsive.

Angular ha sido seleccionado como tecnología principal debido a su arquitectura basada en componentes, su sistema de enrutado, la gestión eficiente del estado y su capacidad para desarrollar Single Page Applications (SPA) escalables y mantenibles.

## Tecnologías
- Angular – Framework principal para el desarrollo de la aplicación. Estructura de las vistas.
- Angular CLI – Herramienta para la creación, compilación y gestión del proyecto.
- TypeScript – Lenguaje utilizado para el desarrollo lógico.
- CSS / Bootstrap – Estilos y diseño visual.
- Node.js & npm – Gestión de dependencias.

## Instalación
npm install en terminal

## Uso
- npm run dev - Modo desarrollador
- npm start - Despliegue
